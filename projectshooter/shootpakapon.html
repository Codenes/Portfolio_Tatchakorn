Project Processing Shooterhuman

รายละเอียด Project

เป็น Games ที่ทำใน Project วิชา Processing เเละได้ทำการขออนุญาติเพื่อนเป็นที่เรียบร้อยเเล้วเพื่อเป็นเเนวทางในการทำ Project ให้เห็นภาพเเละรู้สึกสนิทกับเพื่อนคนนี้มาก ส่วนโปรเจคนี้ทำเกี่ยวกับ การยิงคนเเล้วได้เเต้ม จะมีตัวเราที่เป็นตัวให้เล่นมีการบังคับเมาส์ตัวเรา เมื่อหลบกระสุน ซ้ายเเละขวา เเละจะมีการกำหนดขอบเขตไม่ให้เกินกำหนดเมาส์ X Y เเละกดคลิกซ้ายครั้งเดียวจะเป็นการยิง พอยิงคนล้มจะได้เเต้มมา 1 เเต้ม เก็บสะสมเเต้มได้เรื่อยๆในการยิงคนล้มในเเต่ละนัดที่ยิงได้ ศึกษารายละเอียดเพิ่มเติมได้ที่ไฟล์ PDF
